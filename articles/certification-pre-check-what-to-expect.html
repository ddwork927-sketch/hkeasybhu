認證前評估｜
指明專業人士會睇邊啲位

寬限期登記之後，業主下一步係安排認證。
認證需要由指明專業人士實地視察並簽署核證報告。
本文整理視察通常會睇邊啲位 — 你可以預先準備，避免一次過睇唔過再花錢補。

先講重點

認證 ≠ 簡樸通做嘅嘢

清楚講一次：簡樸通本身唔係指明專業人士。 認證視察、簽署、核證報告、BHU-01 提交，全部由合資格指明專業人士獨立負責。

簡樸通 想做劏房嘅角色，係幫你分流 + 接駁合資格人士， 再協助你整理單位資料 / 工程協調，等專業人士可以更快入場。

通常會睇

1. 最低面積

按官方要求，每間分間單位需符合最低樓面面積。 視察時專業人士會實地度尺，確認單位實際淨面積（一般指內牆對內牆，唔包牆身）。

  • 業主可預先：準備量尺、確保單位內無大件雜物影響量度

2. 最低樓底高度

樓底高度有最低要求。視察時會由地板量到天花。 如有假天花、吊頂、隔音棉，可能會減少有效高度。

  • 業主可預先：知道間單位嘅大概樓底高度
  • 如有 illegal 假天花 → 預早處理

3. 消防安全

視察重點包括：

  • 逃生路徑：每間分間單位嘅通往主出口路徑要暢通
  • 防火物料：分間牆身、門嘅防火等級
  • 滅火 / 警報設備：煙霧探測器、滅火筒（按最新要求）
  • 緊急照明：如有相關要求

4. 通風 + 採光

  • 每間分間單位應有窗或機械通風
  • 採光要求：通常需要自然光或合適照明
  • 抽風扇 / 廚房抽油煙設備（如適用）

5. 結構安全

  • 分間牆嘅承重、固定方式
  • 如有間隔涉及拆除 / 加建：須由結構工程師評估
  • 整層樓宇結構係咪受影響（涉及則師判斷）

6. 排水 + 廁所 + 煮食位

  • 獨立廁所：每間分間單位嘅廁所配置
  • 排水接駁：排污 / 排水有冇接駁正規去水
  • 煮食位：如分間單位內設煮食位，相關通風 / 防火安排

7. 供水 + 獨立水電表

  • 供水：每間分間單位有獨立水喉接駁
  • 水電表：按官方要求，每間單位應有獨立水電表，方便租戶用量計算
業主可以做咩？

視察前嘅 4 件事

  1. 清潔好單位：方便量度同拍照
  2. 準備舊圖則：則師 / 樓書 / 公契附件平面圖
  3. 標好分間單位編號：視察時可清楚對應
  4. 列明已知問題：例如「廁所 1 排水慢」、「房 3 窗戶損壞」 — 對方會更快判斷

視察後通常會收到

  • 初步意見：合格 / 部分整改 / 大規模工程
  • 整改建議清單（如需要）
  • 初步報價（部分專業人士提供）
  • 後續安排（簽署文件 / BHU-01 提交時間表）
