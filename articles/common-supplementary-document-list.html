首頁·文章·8 大常見補件

登記交咗之後
業主最常被要求補咩？

簡樸通 登記支援跟過唔少個案，發現業主最常被要求補嘅資料，都集中喺幾類。
本文整理 8 項，事先準備可以慳成幾星期嚟回。

補件係正常程序

收到補件通知唔等於申請會被拒

收到部門補件通知，多數係文件唔齊或唔夠清晰，唔等於申請會被拒。 但補件需要時間 — 等部門再覆有時要 3–6 星期，影響成個流程。 以下 8 類係實務上最常出現嘅補件項目。

1. 身份證副本唔清晰／過期

業主用幾年前嘅 scan、影印機色差、邊角缺、護照頁面唔齊。 建議：用 iPhone CamScanner / Scanner Pro 重新掃描，PDF 格式，每張獨立檔。

2. 土地查冊過期 / 業權有更新未反映

政府部門通常要求查冊 6 個月內有效。 如業主中間有按揭轉移、聯名變更、業權更動 — 要新一份查冊。

3. 租約簽名 / 印章模糊

舊租約只係影印件嘅影印件，簽名認唔到。 建議：盡量提供有印花稅或律師見證版本；如冇，附補充說明。

4. AR2 同租約嘅單位地址 / 編號唔一致

AR2 寫「太古城 X 座 X 樓」但租約寫「太古城 X 座 X 樓 ROOM A」 — 部門會問返。 業主通常以為一樣，但官方睇文字字面係咪一致。 建議：補一份書面說明，列明 AR2 + 租約 + 實際單位嘅對應關係。

5. 單位相片唔齊 / 唔夠清

常見漏：

  • 有大廳相，無每間分間單位相
  • 影到廁所，但冇影通風窗
  • 影到入口，但無煮食位
  • 整體曝光太暗 / 太背光

建議：補件之前先用 REG-01 文件清單 對返一次。

6. 共同業主同意書未簽 / 唔齊

夫婦聯名但只一個人簽；3 人聯名只 2 人簽 — 部門會卡住等。 建議：開申請前就同所有共同業主溝通好，預早安排簽名／視像見證。

7. 公司持有但無董事決議

有限公司持有物業，業主自己係董事就以為「我簽就得」 — 但部門要求公司層面授權。 建議：起草董事決議書列明授權某董事 / 代理人簽 REG-01。

8. 平面圖 / 草圖唔齊

有業主交咗業權同租約，但完全冇單位平面 / 分間佈局示意圖。 建議：手繪粗略圖都好過冇 — 標出主出口、每間分間單位位置、編號、大概面積。

慳時間嘅 3 個習慣

  1. 所有文件做齊 PDF + 命名清楚：例如「02_身份證_業主A.pdf」、「03_土地查冊_2026-04.pdf」
  2. 每張相加 metadata：iPhone 入面長按相片可以加描述
  3. 補件當日就做：收到部門補件通知，唔好擺到第二、三日先處理 — 響應速度本身會被部門記住
