自己做 vs 搵顧問？

寬限期登記係官方申請程序，業主可以自己做，唔需要顧問。
但每個業主嘅情況唔同 — 有啲人省時省心應該找人，有啲簡單個案自己做完全 OK。
本文用 6 個因素，幫你判斷自己屬邊類。

先設清楚

自己做嘅優劣

優點 劣點
省 HK$1,800 / HK$2,800 顧問費 要熟悉表格、文件、流程
親手控制每個細節 補件 / 跟進都要自己跟
隨時可以問同事 / 朋友 遇到複雜情況容易迷失方向
6 個決策因素

用呢 6 樣嘢評估你嘅情況

① 時間

自己做需時：研究表格 1–2 小時、準備文件 3–8 小時、提交 + 補件跟進另計。 如果你每星期可以擠出 2–3 小時，自己做得切。如果工作好忙、出差多 — 找顧問會慳成個日。

② 業權結構複雜度

  • 個人單獨持有 + 1 個分間單位 → 自己做容易
  • 夫妻聯名 + 雙方都肯簽 → 自己做都 OK
  • 3 人以上聯名 / 公司持有 / 信託代持 → 建議搵顧問，因為授權文件容易出錯

③ 租約 + AR2 狀態

  • 有齊正式租約 + AR2 + 印花稅 → 自己做完全 OK
  • 有租約但無 AR2 → 要稅務協助 + 補充說明 → 建議搵顧問或會計師
  • 租務一筆糊塗 → 建議搵顧問幫你分流

④ 分間單位數量

  • 1–2 間 → 自己做
  • 3–4 間 → 自己做都得，但要花時間整理每間嘅資料
  • 5 間以上 / 多個物業 → 建議搵顧問，避免大量資料管理錯亂

⑤ 中文 / 英文 + 政府表格熟悉度

REG-01 表格本身唔複雜，但部門有時嘅補件通知係官方文書語，需要少少時間理解。 如果你過往做過商業登記、稅務申報、報稅 — 自己做唔難。 如果你少接觸政府表格 — 搵顧問可以避免錯讀補件要求。

⑥ 你嘅性格

  • 鍾意自己控制每個細節 + 唔介意花時間 → 自己做
  • 想有人幫你諗未諗到嘅嘢 + 提醒邊個位易漏 → 搵顧問

典型情境建議

情境 A：自己做啱你

1 個物業、個人持有、1–2 間分間單位、有齊租約 + AR2、過往做過稅務申報、有 5–10 小時時間。 → 自己做完全 OK，慳返 Plan B 嘅 HK$2,800。

情境 B：HK$1,800 認證前初步檢視啱你

仲未決定要唔要做寬限期登記，但想知個單位文件、滲漏水跡象、結構／間隔改動跡象有冇明顯問題。 → Plan A HK$1,800 幫你做風險分流，之後再決定下一步。

情境 C：HK$2,800 登記 + 檢視組合啱你

決定要做寬限期登記，但多個物業、共同業主、公司持有、無 AR2、想一次過搞掂文件同檢視。 → Plan B HK$2,800 由同一個個案經理跟進，唔需要重複交資料。

無論點揀，建議都做嘅 3 件事

  1. 立即備齊文件清單：用 REG-01 文件清單 自己對返
  2. 記低關鍵日期：2026.3.1 開始登記、2027.2.28 截止
  3. 留底所有提交記錄：對外提交嘅每一件文件留 PDF 副本
官方申請渠道 自己做嘅業主，請直接按官方指引提交 REG-01。
