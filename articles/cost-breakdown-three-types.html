劏房合法化費用點計？
3 類費用點分清楚

好多業主問：「總共要幾多錢？」
準確答案要分清 3 類：簡樸通顧問費 + 第三方實費 + 專業及工程費。
本文用 1 個典型個案，拆解每類大概幾錢、邊個收、有冇得唔做。

類別 1

簡樸通顧問費

呢個係簡樸通自己嘅工作費，全部喺 /pricing 列明，無隱藏。

  • A1 文件初步檢查：HK$800（30–45 分鐘 + REG-01 草稿預填）
  • A2 全套登記顧問：HK$2,800–4,800（按情況）
  • A3 全程跟進 + 一次補件：HK$4,800 起
  • B1 分流：HK$2,800（單位資料整理 + 初步分流建議）
  • B2 接駁協調：HK$4,800–8,000（按複雜度）
  • B3 工程整改項目管理：HK$8,000–20,000 起 / 按工程額比例

呢類費用係明牌，唔會因為個案中途加多嘢就突然加價。 如果途中要做超出原本範圍嘅工作，會書面確認後先繼續。

類別 2

第三方實報實銷費

呢類係政府或第三方機構嘅原本費用，簡樸通唔會加價，按實費收 / 業主自付。

  • 土地查冊（每次約 HK$30–80）
  • 公司查冊（每次約 HK$30）
  • 政府申請費 / 認證局申請費（按官方公布為準）
  • 租約印花費（如有）
  • 律師見證授權書 / 同意書嘅費用（如需要）
透明處理： 簡樸通做嘅每筆代付都會留收據 / 截圖。 如果你想自己交，亦可以；簡樸通只負責提示要交咩、邊到交。
類別 3

專業及工程費

呢類由合作工程公司 / 則師 / 工程師直接同業主簽合約收費，簡樸通唔賺差價，亦唔代收。

  • 則師 / 結構工程師視察 + 報告（按單位複雜度，由幾千至幾萬）
  • 消防 / 通風設施改善（按工程量）
  • 裝修 / 拆改 / 防火物料（按報價）
  • 認證視察費（指明專業人士按官方標準收費）

呢類費用差異最大，因為視乎單位本身狀態。同樣係 4 間房嘅劏房， 一個可能只需 HK$3 萬整改，另一個可能要 HK$15 萬。

邊界： 工程公司 / 則師 / 工程師直接向業主負責， 佢哋嘅報價、施工質素、保養、合約糾紛，由佢哋同業主處理； 簡樸通只係轉介 + 跟進。
真實情境

例子：太古城 4 間劏房業主

假設業主有 1 個主樓宇單位、4 間分間單位、想做寬限期登記，未做認證：

  • 簡樸通 A2 全套登記顧問：HK$3,800（包整理、預填、提交、1 次補件）
  • 第三方實費：查冊 HK$50 + 政府申請費（按官方）
  • 專業 / 工程費：登記階段不需要。如果之後要做認證，再睇單位狀況決定。

所以登記階段業主實際付出：大約 HK$3,800 + 實費。認證 / 工程係另一個階段，業主可以分開預算。

想做劏房想做新劏房嘅情境會唔同 — B1 分流 HK$2,800 之後， 要視乎則師判斷是否可行。如果可行，工程費可能由幾萬到十幾萬。 簡樸通會幫你做事前預算範圍，避免落定先發現超預算。

邊類費用唔做得？

  • 顧問費：業主可以自己處理文件 — 唔需要顧問。但要有時間同熟悉表格。
  • 實費：如政府申請費 / 認證局費，唔可以唔交。
  • 專業及工程費：如果單位需要整改先可認證，呢類無得避。
官方收費權威來源 政府申請費、認證費按官方公布為準；本文只屬整理。
